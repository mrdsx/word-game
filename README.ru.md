# Classic word game

Classic word game - это облачная онлайн игра.

## Локальная установка проекта

### Что нужно перед установкой

Список зависимостей для каждого сервиса перед его установкой:

- Фронтенд: bun (пакетный менеджер)
- Бэкенд: uv (пакетный менеджер)
- Firebase: docker + docker compose

### Фронтенд

Сначала скопируйте переменные окружения из `.env.dev` или `.env.prod` в файл `.env` для локальной разработки или продакшена соответственно. Пустые переменные окружения нужно заполнить самому.

Далее запустите следующие команды:

```bash
bun i --frozen-lockfile
bun dev
```

### Бэкенд

Сначала скопируйте переменные окружения из `.env.dev` или `.env.prod` в файл `.env` для локальной разработки или продакшена соответственно. Пустые переменные окружения нужно заполнить самому.

Далее запустите следующие команды:

```bash
uv sync --frozen
uv run uvicorn src.main:app --reload
```

### Firebase

Используйте docker compose для запуска эмуляторов Firebase, указав `firebase` как единственный контейнер для запуска:

```bash
docker compose up firebase
```
